# Fine Tuned Language Classifier
### Introduction
This project was a homework for class. The goal was to fine tune a
pre-existing text classifier so that it would be more adapt to a specific classifying task.
The model used was the distilbert-base-uncased from Hugging Face and the data was from a 
previous study on determining if text could be classified as written by human or generative ai.
